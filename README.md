# IT Ticket Support Management System Analysis (Excel)

## Objective
To analyze IT support ticket data and evaluate service efficiency, agent performance, and customer satisfaction, with the goal of identifying operational bottlenecks and improvement opportunities.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Advanced formulas
- Charts & Interactive Dashboards
- Slicers and filters

## Dataset Overview
The dataset contains 5 years of IT support ticket data, including ticket severity, resolution time, request category, agent details, and satisfaction ratings.

## Key Analysis Performed
- Ticket volume trends over time (Year / Quarter)
- Agent-wise efficiency and performance scoring
- Resolution time analysis by severity and category
- Satisfaction rating analysis across demographics
- Identification of high-load and underperforming segments

## Dashboard
An interactive Excel dashboard was built to track KPIs such as:
- Ticket volume
- Average resolution time
- Satisfaction rating
- Severity distribution
- Request category trends

## Business Recommendations
- Optimize agent allocation based on workload and efficiency
- Focus training on high-severity and low-satisfaction segments
- Improve response workflows for frequently delayed ticket categories

## Files in This Repository
- `it_ticket_analysis.xlsx` – main analysis file
- `dashboard.xlsx` – interactive dashboard
